Un script JS para realizar Test mediante archivos .Json.

1. Crea un test en formato .json siguiendo el formato del archivo prueba.json que viene incluido en el script.
2. Selecciona el fichero .json que tenga el test que quieres hacer.
3. Pulsa en iniciar test para comenzar
4. Una vez finalizado, pulsar en reiniciar test para volver a seleccionar el fichero .json para volver a hacer el test (o uno diferente)
